# 📈 Stock Price Prediction using LSTM

A deep learning project that predicts future stock prices using historical market data and Long Short-Term Memory (LSTM) networks. The model learns patterns from past stock prices to generate accurate time-series predictions.

---

## 🚀 Features

- Predicts future stock prices using LSTM networks.
- Performs data preprocessing and feature scaling.
- Uses historical stock market data for training.
- Visualizes actual vs predicted stock prices.
- Evaluates model performance using appropriate metrics.

---

## 🛠️ Tech Stack

- **Programming Language:** Python
- **Framework:** TensorFlow
- **Libraries:** Pandas, NumPy, Matplotlib, Scikit-learn

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/sharanya-1521/stock-price-prediction-lstm.git
```

Navigate to the project directory

```bash
cd stock-price-prediction-lstm
```

Install the required dependencies

```bash
pip install tensorflow pandas numpy matplotlib scikit-learn
```

Run the project

```bash
python stock_prediction.py
```

## 📊 Workflow
